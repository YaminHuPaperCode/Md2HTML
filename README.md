# Md2HTML
#### __Md2HTML is a converter from markdown to HTML which supports:__

 - headers 
 - paragraphs
 - emphasis
 - strong emphasis
 - strikethrough
 - underline
 - nested emphasizing of text
 - code representation
 - shielding
 

P.S. Md5Html is a full version. Md5HTMLUnderline has some bugs, don't use it. :)
